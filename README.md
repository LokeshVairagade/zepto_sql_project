# Zepto SQL Project

## 📖 Overview
Zepto SQL Project ek database-driven application hai jo SQL queries aur relational database concepts ko demonstrate karta hai. Iska main objective hai CRUD (Create, Read, Update, Delete) operations ko implement karna aur real-world scenarios ke liye database management sikhana.

## 🎯 Objectives
- SQL fundamentals ko practically apply karna
- Relational database design aur normalization
- CRUD operations ka implementation
- Joins, subqueries, aur aggregate functions ka use
- Real-world case study (e.g., e-commerce, library management, student records)

## 🚀 Features
- User-friendly schema design
- Multiple table relationships (1:1, 1:N, M:N)
- Sample dataset for testing
- Optimized queries for performance
- Error handling aur constraints (Primary Key, Foreign Key, Unique, Not Null)
